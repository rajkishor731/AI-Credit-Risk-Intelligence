# AI Credit Risk Intelligence Platform

An AI-powered credit risk analysis system that analyzes company financial reports, news sentiment, and risk signals to assist in automated credit decision making.

## Features

* Extract financial data from PDF reports
* Calculate financial ratios
* Analyze company news
* Detect financial risk signals
* AI-based risk interpretation
* Credit scoring system
* Loan approval decision
* Automatic CAM report generation
* Interactive dashboard using Streamlit

## Tech Stack

* Python
* Streamlit
* Plotly
* OpenAI API
* News API
* PDF extraction

## Run the Project

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py
